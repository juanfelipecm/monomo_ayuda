# De proyecto a contrato

> This is a template page

Crear un proyecto en Monomo es fácil y rápido. A continuación, te explicamos cómo hacerlo en pocos pasos.

### Paso 1: Accede a la plataforma

- Inicia sesión en tu cuenta de Monomo. Si no tienes una, regístrate en la página de inicio.

### Paso 2: Navega hasta "Crear Proyecto"

- En el menú principal, selecciona "Crear Proyecto". Aquí es donde iniciarás el proceso de creación.

### Paso 3: Elige la categoría de tu proyecto

- Define la categoría que mejor se ajusta a tus necesidades: reseñas de productos, unboxing, GRWM (Get Ready With Me), tutoriales, etc.

### Paso 4: Detalla el brief

- Explica claramente qué tipo de contenido necesitas. Sé específico sobre lo que esperas de los creadores (duración, estilo, tono, etc.) y el producto o servicio que deseas promocionar.

### Paso 5: Establece un presupuesto

- Define el presupuesto para el proyecto. Monomo te permitirá elegir entre diferentes tarifas para los creadores, y puedes ajustar tu oferta dependiendo del tipo de contenido.

### Preguntas frecuentes:

- **¿Puedo crear varios proyectos al mismo tiempo?** Sí, puedes gestionar varios proyectos simultáneamente desde tu panel de control.
- **¿Qué hago si no sé cómo redactar el brief?** Monomo tiene plantillas para facilitarte el proceso.
- **¿Puedo cambiar la categoría del proyecto una vez creado?** No, pero puedes editar otros detalles del proyecto si es necesario.
