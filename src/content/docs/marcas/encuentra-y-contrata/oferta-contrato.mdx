# **De oferta a contrato**

> Es hora de que revisemos como podrás enviar ofertas a los talentos 😎
> 

## **¿Cómo funcionan las ofertas en Monomo?**

Dentro del perfil de cada modelo, podrás enviar una propuesta inicial, dónde se detalla la información del contenido solicitado y el valor del proyecto. Estas ofertas podrás administrarla tanto del proyecto, como del inbox en el apartado de ofertas.

### **Estado de las ofertas**

**🔵 Esperando respuesta:** Ofertas que has enviado pero aún no tienen respuesta.

**🟢 Aceptada:** Ofertas que los modelos han aceptado.

**🟡 Negociación:** Cuando el modelo hace una contraoferta a la propuesta enviada.

**🔴 Rechazada:** Propuestas que han sido declinadas por ti o por el modelo.

**🔍 Importante:** Que el modelo acepte una oferta no implica que el contrato se formalice de inmediato. Siempre se requiere la confirmación final de tu parte. Tu podrás abrir un chat con el modelo para ajustar detalles antes de confirmarlo.

## **De oferta a contrato**

Una vez que confirmas el contrato, el chat de ofertas se traslada a la sección de contratos. Aquí, se habilitará un acceso directo al contrato, que será el espacio central para gestionar todo el proyecto y donde el modelo subirá el contenido creado.

### **¿Qué incluye el contrato?**

**📁 Nombre del Proyecto:** El título del proyecto en el que trabajarás.

**🎮 Tipo de Proyecto:** Especifica si es un reel, TikTok o presencial.

**🔍 Visualiza el Proyecto:** Botón para acceder al proyecto y revisar la propuesta completa de la marca.

**🖍️ Release / Cesión de Derechos:** Detalla la autorización que te da el modelo para el uso de su imagen en el contenido creado.

**❓ Sección de Ayuda:** Revisa las preguntas frecuentes para resolver tus dudas.

**📼 Contenidos de Contratación:** Aquí es donde el modelo subirá el contenido solicitado.

> ‼ Recuerda: Siempre manté todas las conversaciones con los creadores dentro de Monomo. Esto garantiza que el proyecto fluya de manera eficiente y dentro de los términos acordados.
