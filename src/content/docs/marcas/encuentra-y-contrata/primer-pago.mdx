# **Realiza tu primer pago**

> Sabemos lo importante que es realizar el pago para poder descargar el contenido y queremos asegurarnos que tengas toda la info necesaria 🔥
> 

Una vez que hayas aprobado el contenido, podrás realizar el pago a través de nuestra plataforma para descargar el material y finiquitar el contrato.

# Todo sobre Stripe 🔧

Stripe es nuestra pasarela de pagos aliada donde podrás gestionar tus suscripciones y realizar los pagos de tus contratos de manera sencilla y segura. Aquí te contamos cómo empezar y qué debes tener en cuenta 🤓👆

## Detalles importantes a tener en cuenta:

**💰 Comisión por contrato:** En Monomo aplicamos una comisión del 10% sobre el valor de cada contrato.

**💳 Requisitos de pago:** El pago debe realizarse en su totalidad antes de poder descargar el contenido aprobado. Este puede ser con tarjeta de **débito** y/o **crédito**.

**🔒 Seguridad:** Stripe garantiza transacciones seguras para proteger tus datos y pagos.

## Cómo realizar tu pago:

1. Ingresa a la plataforma y dirígete a la sección de contratos.
2. Selecciona el contrato correspondiente al contenido aprobado.
3. Haz clic en “Realizar pago”.
4. Completa los datos de tu tarjeta de crédito o débito.
5. Confirma la transacción y espera la notificación de confirmación.

Una vez completado el pago, el contenido estará disponible para descarga inmediata.
