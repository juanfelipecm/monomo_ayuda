# Como funciona un contrato

> Ahora que tienes tu primer contrato, vamos a explorar cómo funciona 🤓
> 

# ¿Qué es un contrato?

El contrato es el espacio donde podrás subir el contenido para la marca, recibir retroalimentación y gestionar el cobro. Es crucial que todo el contenido se cargue directamente a través del contrato en Monomo y no por medios externos. Esto asegura que el proceso se mantenga organizado y seguro.

## Partes del contrato


**📁 Nombre del Proyecto:** El título del proyecto en el que estás trabajando.

**🎬 Tipo de Proyecto:** Especifica si es un reel, TikTok o presencial.

**🔍 Visualiza el Proyecto:** Tendrás un botón para ver el proyecto y leer la propuesta de la marca.

**📝 Release / Cesión de Derechos:** Aquí encontrarás la autorización que permite a la marca usar tu imagen.

**❓ Sección de Ayuda:** Consulta la sección de preguntas frecuentes para resolver cualquier duda.

**📤 Contenidos de Contratación:** Esta es la sección donde podrás subir el contenido relacionado con el proyecto.
